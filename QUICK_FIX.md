# ⚡ Швидке виправлення білого екрану

## Проблема
Білий екран на Vercel, але локально все працює.

## Швидке рішення

### Варіант 1: Тимчасово вимкни middleware (для тесту)

1. Перейменуй `middleware.ts` → `middleware.ts.disabled`
2. Commit і push
3. Redeploy
4. Перевір чи сайт працює

**Якщо працює** - проблема в middleware, потрібно виправити логіку.

### Варіант 2: Спрости matcher

Якщо matcher не працює правильно, спробуй спростити:

```typescript
export const config = {
  matcher: ['/((?!api|assets|_next|.*\\..*).*)'],
};
```

### Варіант 3: Перевір чи rewrites працюють

1. Відкрий `develop.ways2spain.com/index.html` напряму
2. Якщо завантажується - rewrites працюють ✅
3. Якщо ні - проблема в Vercel конфігурації

### Варіант 4: Перевір Network tab

1. DevTools → Network
2. Онови сторінку
3. Шукай запити до:
   - `/index.html`
   - `/assets/index-*.js`
   - `/assets/index-*.css`

**Якщо є помилки (404, 403):**
- Middleware блокує файли
- Або rewrites не працюють

**Якщо запитів немає взагалі:**
- React не завантажується
- Проблема в index.html

## Найімовірніша причина

Middleware перехоплює запити до статичних файлів ДО того як Vercel rewrites можуть їх обробити.

**Рішення:** Переконайся що matcher виключає статичні файли, а middleware пропускає їх на початку.
